## Configuration

Notes:

* Install Node.js
* Set up cloud functions
* Set up Firebase
* Set up STMP email service
* Install firebase email extension
 * Create new Gmail account
 * Enable 2FA
 * Create App Password
 * Visit accounts.google.com/b/0/DisplayUnlockCaptcha
* Set up Cloud billing
* Set up .env db tokens
